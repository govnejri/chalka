### 2.2 Big O и базовые структуры

Фокус
- асимптотика O(1), O(log n), O(n), O(n log n), O(n²)
- анализ по времени и по памяти
- разница массив против списка на уровне операций

Темы
- оценка циклов и вложенных циклов
- стоимость вставки и удаления в разных структурах
- когда линейный проход лучше бинарного поиска

Задачи
- [ ] Для набора псевдокодов прикинуть сложность и объяснить на словах
- [ ] Написать три реализации поиска минимума: один проход, сортировка, два прохода. Замерить время на больших массивах
- [ ] Взять 5 произвольных задач с LeetCode easy и для каждой выписать сложность своего решения

### 2.3 Линейные структуры

Фокус
- массивы
- связные списки
- стек
- очередь
- двухсторонняя очередь

Темы
- операции push, pop, insert, delete
- реализация стеков и очередей через список и через связный список
- цикл по индексу против цикла по итератору

Практика и задачи
- [ ] Реализовать односвязный список с методами `append`, `prepend`, `delete`, `find`
- [ ] Реализовать стек и очередь на списках Python
- [ ] Two Sum (Arrays & Hashing)
- [ ] Contains Duplicate
- [ ] Valid Palindrome (Two Pointers)
- [ ] Reverse Linked List

### 2.5 Деревья и графы

Фокус

- структура узла дерева

- [ ] Invert Binary Tree

### 2.6 Сортировки и бинарный поиск

Фокус
- понимание внутренних сортировок
- умение узнавать бинарный поиск в задачах

Темы
- простые сортировки: выбор, вставка, пузырек (для тренировки)
- быстрые: merge sort, quicksort на уровне идей
- бинарный поиск по массиву и по ответу

Практика и задачи
- [ ] Реализовать merge sort и quicksort
- [ ] Написать чистый бинарный поиск по отсортированному массиву
- [ ] Binary Search (classic)
- [ ] Search Insert Position
- [ ] Search in Rotated Sorted Array
- [ ] Find Minimum in Rotated Sorted Array

### 2.7 Ежедневная LeetCode практика

Режим
- 1–2 задачи в день
- сначала easy, затем medium
- писать решение
- потом смотреть варианты напарника и разборы

Сетка по темам
- [ ] Arrays & Hashing: Two Sum, Contains Duplicate, Product of Array Except Self, Valid Anagram
- [ ] Two Pointers: Valid Palindrome, 3Sum, Container With Most Water
- [ ] Sliding Window: Best Time to Buy and Sell Stock, Longest Substring Without Repeating Characters
- [ ] Stack: Valid Parentheses, Min Stack, Daily Temperatures
- [ ] LinkedList: Reverse Linked List, Merge Two Sorted Lists, Linked List Cycle, Reorder List
